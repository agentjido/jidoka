# Changelog

## v0.1.0

Initial release candidate.

- Added durable session, run, attempt, artifact, verifier, outcome, and event
  structs.
- Added the public `Jidoka` and `Jidoka.Agent` facades for session and run
  lifecycle operations.
- Added execution and verifier adapter boundaries.
- Added the Jido AI prompt execution adapter and controlled workspace tools.
- Added the CLI prompt, chat, and MVP evaluation commands.
- Added deterministic MVP fixtures for approve, retry, and resume flows.
